4.2 Usage

You compile your Reach code by executing

  reachc SOURCE EXPORT ...

where SOURCE is your source file, and each EXPORT is an exported Reach.App. If no EXPORT is provided, then main is used.

reachc supports the following options:

  • -o/--output ‹OUTPUT› — Writes the compiler output files to ‹OUTPUT›, which is the current directory by default.

  • --intermediate-files — Write intermediate files, which may be interesting for debugging compilation failures or using in other contexts.
