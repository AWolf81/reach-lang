3.15 IDE/Text Editor Support

There are a number of text editor plugins available to provide a better development experience for Reach. These plugins provide syntax highlighting, build commands, key mappings, snippets, and more. The following plugins are availabe:

  • Visual Studio Code: Reach IDE

  • Atom: Reach Language Support, Reach IDE

  • Sublime Text: Reach IDE
