1.1 Installation & Usage

Reach is a Dockerized program, so it has no dependencies and has a trivial installation process. You can install it by running:

  curl https://raw.githubusercontent.com/reach-sh/reach-lang/master/reachc -o reachc ; chmod +x reachc

in your project repository.

After you install it, you can compile your Reach code by executing reachc SOURCE where SOURCE is your executable source file. reachc supports the following options:

  • -o/--output ‹OUTPUT› — Writes the compiler output files to ‹OUTPUT›, which is the current directory by default.
